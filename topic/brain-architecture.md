---
title: Brain Architecture
---
The content repeatedly explains what a brain is and how it is structured as a modular AI-powered knowledge agent. A brain combines a reusable brain model with instance-level configuration in `brain.yaml` to produce a running system. The architecture emphasizes separation between the model, the lightweight instance package, and the runtime shell. This model supports personal, team, and community-oriented deployments while keeping the same core runtime. The docs also frame brains as independently deployable MCP servers with identity, content, capabilities, and interfaces. This is the central conceptual backbone across the batch.
