---
title: Entity Types
---
The content explains how brains represents durable knowledge as typed markdown entities. It outlines the core idea that each entity type is registered by a plugin, validated with Zod, indexed for search, exposed through system tools, and sometimes rendered by the site builder. The reference also describes how files in `brain-data/` map to entity types through directory sync. This makes entity typing a central part of how the runtime organizes knowledge. The documentation is broad and serves as the main reference for understanding what kinds of entities exist and how they are handled.
