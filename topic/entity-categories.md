---
title: Entity Categories
---
The reference groups entity types into several functional categories, including identity, core knowledge, derived knowledge, publishing, marketing, products, and agent directory entities. Each category contains specific types such as `brain-character`, `anchor-profile`, `base`, `prompt`, `topic`, `site-info`, `post`, `product`, and `agent`. For many of these, the document lists key fields, singleton behavior, and availability across different presets. This taxonomy is useful because it shows how the runtime separates knowledge, content production, and publishing workflows. The breadth of the catalog makes this one of the document's main themes.
