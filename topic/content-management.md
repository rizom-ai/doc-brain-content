title: Content Management
---
Another core topic is the markdown-based content model used by brains. Durable knowledge is stored as typed entities with YAML frontmatter and markdown bodies, and those entities can be created, updated, searched, and extracted through shared system tools. The documentation covers directory sync, CMS editing, generation workflows, publishing statuses, image references, and prompt customization. It also describes how entity types like notes, links, posts, decks, newsletters, topics, and summaries fit into the same graph. This topic is central because content is the system's source of truth and the basis for search, publishing, and agent context.
